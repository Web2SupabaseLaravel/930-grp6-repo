# Laravel + Supabase API Project

This is a Laravel-based web application integrated with **Supabase** as the backend database.  
The project manages admin access to both humans and tickets using defined API endpoints.

---

## 🚀 Features

- Admin can assign access to humans and ticket types  
- RESTful API structure  
- Connected to Supabase PostgreSQL  
- Structured MVC using Laravel  
- API tested via Postman  

---

## 📁 API Endpoints

### Human Access
- `GET /api/admin/humans` – List all human access  
- `POST /api/admin/humans` – Add new human access  

### Ticket Access
- `GET /api/admin/tickets` – List all ticket access  
- `POST /api/admin/tickets` – Add new ticket access
